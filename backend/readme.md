## heroku command

```
heroku login
heroku create
git remote remove heroku
git remote add heroku https://git.heroku.com/gitman-restapi.git
git push heroku main
```

## Starting the app

```
npm install
npm start
```

## pushing to web shortcut

```
git add . && git commit -m "terminal" && git push heroku main && git push origin main
```
